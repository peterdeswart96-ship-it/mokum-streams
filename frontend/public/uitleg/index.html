Handmatig starten & stoppen

Hoe een stream start & stopt

Van één klik in het dashboard tot een live uitzending op YouTube — welke onderdelen er meepraten, in welke volgorde, en waar de veiligheidsnetten zitten.

De onderdelen

Acht schakels werken samen. De kern van het ontwerp: het dashboard vraagt niks rechtstreeks aan OBS — het zet commando's in een wachtrij, en de agent op de OBS-pc voert die uit. Zo hoeft de OBS-pc geen open poorten naar buiten te hebben.

01Een stream starten

  1. Je klikt "+ Nieuwe stream" en kiest tafel, titel en zichtbaarheid (Openbaar / Verborgen / Privé).
  2. Het dashboard stuurt dit naar de backend — met je toegangscode als bewijs dat je mag bedienen.
  3. De backend zoekt de vaste streamsleutel van die tafel op (uit de opslag). Elke tafel heeft een eigen, herbruikbare sleutel.
  4. De backend maakt een YouTube-broadcast aan en koppelt 'm aan die streamsleutel. De broadcast staat nu klaar ("Upcoming").
  5. De backend markeert de tafel als bezet en zet een startStream-commando (+ de overlay-standen) in de wachtrij.
  6. De agent pollt de wachtrij (elke 3 seconden) en pikt het startStream-commando op.
  7. De agent zegt tegen OBS: begin met zenden — of, met de #41-fix, forceert 'ie een schone stil→zenden-flank als OBS al zou zenden.
  8. OBS combineert camerabeeld + overlays en stuurt dat naar de streamsleutel.
  9. YouTube ziet de stil→zenden-overgang en zet de broadcast LIVE — kijkers zien vers beeld.
  10. De agent bevestigt dat het commando verwerkt is; de backend haalt het uit de wachtrij.

02Een stream stoppen

  1. Je klikt "Stop stream" op de tafelkaart.
  2. Het dashboard stuurt dit naar de backend.
  3. De backend zet een stopStream-commando in de wachtrij en markeert de tafel weer als vrij, zodat je 'm opnieuw kunt gebruiken.
  4. De agent pikt het commando op bij de volgende poll (binnen 3 sec).
  5. De agent zegt tegen OBS: stop met zenden.
  6. OBS stopt → YouTube ontvangt geen beeld meer → de broadcast eindigt automatisch.

De vangnetten

Het systeem draait onbemand op een pc in de zaal, dus er zitten bewust veel veiligheidsnetten in. Ze vallen in vier groepen: dat er nooit iets blíjft hangen, dat er geen dubbele of vastgelopen acties ontstaan, dat de agent overeind blijft, en dat er niks per ongeluk automatisch gebeurt.

Er blijft nooit iets hangen

Nacht-stop (02:00)

Stopt élke nog-lopende stream 's nachts — ook handmatig gestarte — zodat er niks blijft doorzenden met een bevroren beeld. Staat los van de master-schakelaar, dus werkt altijd.

timer · elke 30 min

Auto-stop

Een ingepland toernooi stopt vanzelf zodra Cuescore het als "Finished" meldt, mét de geplande eind-tijd als extra vangnet (ook als Cuescore even onbereikbaar is).

Schone flank (#41)

Zond OBS al? Dan forceert de agent een verse stil→zenden-start i.p.v. een oud, bevroren beeld door te sturen — precies wat op 14-07 misging.

agent · startStream

Geen dubbele of vastgelopen acties

Tafel al in gebruik

De backend weigert een tweede stream op een tafel die al bezet is (foutmelding i.p.v. chaos). Eerst stoppen, dan opnieuw starten.

HTTP 409

Eén broadcast per tafel per dag

De automatische aanmaak is idempotent: een tafel krijgt niet per ongeluk twee broadcasts, ook niet als een timer twee keer draait.

broadcasts-opslag

Retry-guard (35s)

Wordt een startcommando dubbel afgeleverd (na een mislukte bevestiging), dan herstart de agent een gezonde, net-gestarte stream niet.

agent

De agent blijft overeind

Loop-proof

Een commando dat nóóit kan slagen (bijv. een bron die niet bestaat) wordt gedropt i.p.v. eeuwig herprobeerd. Zo'n lus legde de agent ooit plat — nu niet meer.

SOURCE_NOT_FOUND

Herverbindt vanzelf

Na een OBS-herstart maakt de agent bij het volgende commando automatisch een verse verbinding — geen handmatige herstart nodig.

obs-websocket

Herstart-taak

De agent draait als Windows-taak die zichzelf herstart bij een crash én meestart bij het opstarten van de pc.

Task Scheduler

Niks gebeurt per ongeluk automatisch

Master-schakelaar

De hele auto-motor (starten én stoppen van geplande toernooien) staat achter één schakelaar, standaard uit. Handmatig werkt altijd; automatisch pas als we 't bewust scherpzetten.

AUTOMATION_ARMED
